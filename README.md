# SKALA-FRONT

HTML/CSS 학습 내용과 과제를 프로젝트 단위로 구분한 저장소입니다.

## 폴더 구조

```text
.
├── assignment/
│   ├── day1/          # 여행 사이트 과제
│   └── day2/          # 반응형 상품 카드 과제
└── practice/
    ├── personal-portal/     # 개인 페이지를 모은 다중 페이지 사이트
    │   ├── index.html
    │   ├── *.html
    │   └── assets/
    └── profile-portfolio/   # 개인 프로필 포트폴리오
```

각 폴더는 서로 독립적인 결과물입니다. 실행할 때는 해당 폴더의 `index.html`을 시작 페이지로 사용합니다.
